# SAS Daily Practice Repository

This repository documents my **structured daily practice in SAS programming**, focusing on data handling, analysis, and reporting workflows commonly used in real-world analytical environments.  
All programs are written with clarity, reproducibility, and good programming practices in mind.

---

## 📌 Objective
- Develop strong foundations in **SAS DATA step and PROC step**
- Practice **data manipulation, summarization, and analysis**
- Understand **statistical outputs and analytical logic**
- Build clean, reusable, and well-documented SAS programs
- Maintain a long-term learning archive for revision and improvement

---

## 📂 Repository Organization

Each **Day folder** reflects the concepts practiced on that day, and file names clearly describe the logic implemented.

---

## 🧠 Concepts Practiced

### 🔹 Data Handling
- DATA step programming
- IF–ELSE logic
- WHERE vs IF conditions
- Variable creation and transformation
- Character and numeric functions
- Date handling and formatting

### 🔹 Data Inspection and Reporting
- PROC PRINT
- PROC CONTENTS
- PROC SORT
- BY-group processing

### 🔹 Categorical Data Analysis
- PROC FREQ
- Frequency distributions
- Percent and cumulative percent
- Handling missing values
- Order and formatting effects

### 🔹 Summary Statistics
- PROC MEANS
- PROC SUMMARY
- Descriptive statistics (N, Mean, Median, Min, Max, Std)
- Group-wise summarization

### 🔹 SQL-Based Analysis
- PROC SQL SELECT statements
- COUNT and DISTINCT logic
- GROUP BY and ORDER BY
- Use of PUT() for formatted grouping

### 🔹 Formats and Presentation
- FORMAT statement
- PUT function
- Date formats (DATE, MONNAME, MONYY)
- Understanding display vs stored values
